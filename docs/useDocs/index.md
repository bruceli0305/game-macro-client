# Game Macro 使用手册

欢迎使用 Game Macro 游戏宏自动化工具！本手册专为普通用户设计，帮助您快速上手使用这个强大的游戏辅助工具。

## 🚀 快速开始

### 选择使用方式

#### 方式一：直接运行版本（推荐新手）
**适合人群**：普通用户，不想安装额外软件
**特点**：下载即用，无需安装 AutoHotkey

**安装步骤**：
1. 从发布页面下载 `game-macro-vX.X.X.zip` 压缩包
2. 解压到您电脑上的任意文件夹
3. 双击运行 `GameMacro.exe` 文件启动程序

#### 方式二：源码版本（适合高级用户）
**适合人群**：熟悉 AutoHotkey 的用户，需要自定义功能
**特点**：需要安装 AutoHotkey，可以修改源码

**安装步骤**：
1. 确保您的电脑已安装 AutoHotkey v2.0 或更高版本
2. 下载源码版本的压缩包
3. 解压到您选择的目录
4. 双击运行 `Main.ahk` 文件启动程序

### 基本设置（两种方式通用）
- 首次启动时选择界面语言（中文/英文）
- 设置您要使用的游戏路径
- 配置您习惯的快捷键
- 使用内置的技能编辑器创建宏

## ❓ 常见问题

### 安装问题
**Q: 我应该选择哪种版本？**
A: 新手用户推荐使用直接运行版本（exe文件），无需安装额外软件。熟悉 AutoHotkey 的用户可以选择源码版本进行自定义。

**Q: 直接运行版本无法启动？**
A: 请确保您的系统是 Windows 10 或更高版本，并且解压了完整的压缩包文件。

**Q: 源码版本无法启动？**
A: 请确保已安装 AutoHotkey v2.0 或更高版本，并且所有源码文件完整。

**Q: 启动后没有反应？**
A: 检查系统托盘区域，程序可能已最小化运行。

### 使用问题
**Q: 宏不执行怎么办？**
A: 请确保游戏窗口处于激活状态，并且快捷键设置正确。

**Q: 如何查看运行状态？**
A: 程序界面会显示当前运行状态和日志信息。

### 性能问题
**Q: 程序占用资源太多？**
A: 可以调整检测频率和优化设置来降低资源占用。

**Q: 两种版本性能有差异吗？**
A: 直接运行版本经过优化，启动更快。源码版本更适合需要自定义功能的用户。
